# Designs — Marketing Visuals

Self-contained HTML design assets. Each opens in any browser (the source artifact) and can be sent to **Adobe Express** (`export_html_to_express`) or imported into **Canva** (import design from URL) for final edits and one-click export to PNG/JPG.

Briefs and brand kit: [`../03-content/design-briefs.md`](../03-content/design-briefs.md).

All five open in a browser (the source artifacts) and are also live as editable Adobe Express docs (export PNG/JPG/PDF from there). To re-export after editing an HTML file: `html_export_readiness_skill` → `export_html_to_express`.

## Fonts
Bebas Neue (display) + Roboto (body) via Adobe Fonts kit `hmj5gxj`, embedded in each file's `<head>`. Internet connection needed to render the kit.

## How to export
1. Preview: open the `.html` in a browser.
2. To Adobe Express: validate (`html_export_readiness_skill`) → `export_html_to_express` → edit/export in Express.
3. To Canva: host the HTML or use Canva's import-from-URL, or just screenshot at 1:1 for a quick post.

## To edit "you asked, we built it"
Open `asset-3-you-asked.html`, change the text inside `.req` (the request) and `.built` (what shipped). Re-export. That's the whole loop, made visible.
