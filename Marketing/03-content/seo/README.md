## How to use
1. **Publish** on your site/blog (one article = one URL = one query). Set canonical URLs.
2. **Repurpose each** into: 1 LinkedIn carousel + 1 community answer + 1 dev.to/Hashnode repost (canonical back to your site). Write once, post 5×.
3. **Cross-link** articles to each other and to the free lab CTA already inside each.
4. **Track** which articles bring activated learners in `01-action-plan/targets-and-metrics.md` (SEO row).

## Timeline reality
SEO compounds slowly — expect traction in 2–3 months (Phase 2–3). Publish now so the clock starts. GEO (AI citations) can land faster if the answer chunks are clean, which these are.
