# My QA Project

My personal Quality Assurance knowledge base — the notes, roadmaps, and
reference material I'm using to go from beginner to job-ready QA / automation
tester, plus the early plan for a QA learning platform I'm building.

## The two roadmaps

1. **Manual QA** — Foundation → Core QA Skills → Specialized Testing → Real
   Projects & Job Prep, ending interview-ready.
2. **Selenium + Java automation** — Java essentials → WebDriver → TestNG + Page
   Object Model → Maven / reporting / CI-CD → end-to-end project. Stack: Java 17,
   Maven, TestNG, ChromeDriver, ExtentReports/Allure, Log4j2, GitHub Actions.
